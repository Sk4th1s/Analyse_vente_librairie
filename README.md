# Analyse_vente_librairie

Analyser les ventes d'une librairie

Dasn ce projet, j'ai commencé par regarder le chiffre d'affaires par trimestre pour voir son évolution. Ensuite j'ai regarder le chiffre d'affaires par catégorie et les tops/flops des ventes et du chiffre d'affaires. J'ai continué avec une analyse sur les clients pour voir si il fallait cibler une catégorie de personne (ex: les personnes entre 40 et 50ans), avec les paniers moyens par âge, la fréquence d'achat et ensuite le montant total des achats. J'ai fini avec une analyse sur les catégories les plus achetées par âge.
